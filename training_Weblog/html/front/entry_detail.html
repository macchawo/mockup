2016JavaOne報告会に行ってきた
2016.03.12
Java
先日、関西Javaエンジニアの会が主催する「2016JavaOne報告会」に参加しました。
サンフランシスコで開催された2016JavaOneのセッションの紹介などがあり、「自分ももっと勉強せなあかんな！」という気持ちになりました。

2016JavaOneでは、Java EEに関するセッションはほとんどなく、Java SEに関するセッションが多数を占めたそうです。
その中でもJava SEのチーフアークテキトのMark Reinholdが率いるProjectJigesawに注目しました。
時期Java SE9で導入予定の機能となりますが、モジュールによるClassPathの管理ができるようになり、標準ライブラリーもモジュールに分割して、アプリケーションが必要なモジュールだけを選択できるようになります。

Java EEに関しては、大きな進展はなかったそうですが、Java EEの動向に注目して、参画するプロジェクトで活用できるようにしていきたいと考えています。
ちなみに、先日「Java EE7徹底入門」が出版されました。Java EE7に関する紹介がされていますので、注目です。
